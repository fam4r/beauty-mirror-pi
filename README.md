# Carducci-Mirror

by famar and nicolasfara for FabLabRomagna

Using pygame, made for Raspberry Pi.
Tested on Raspberry Pi 3.

Requirements:
- Raspberry Pi 3
- python3
- pygame
- PIR sensor

Getting started:
```shellsession
git clone https://github.com/crash-it/carducci-mirror.git
cd carducci-mirror
python3 smartmirror.py
```
